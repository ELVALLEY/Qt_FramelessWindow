## 借鉴于Github实现的Qt无边框;



### 实现的功能有:

1. 支持通过使用鼠标拖拽来改变窗体位置和大小；

2. 支持双击标题栏最大化窗体和还原窗体；

3. 支持自定义边框阴影；
4. 支持跨不同DPI的屏幕拖拽；

5. 适应分辨率和DPI改变；

### 未完成功能:

1. 支持`Windows Areo Snap`特性；
2. 支持系统阴影;
